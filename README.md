# trash
Basic linux command line trash utility. 

Gives access to restore functionality, tested on KDE but should work fine with Gnome as well. 

# Use

  trash <file|folder|pattern>...
  trash -list
  trash -empty
  trash -restore <filename>
  trash -restore-all

Options:
  -f, --force     ignore missing paths/patterns
  -h, --help      show help
